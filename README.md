# ProjectH

利用 UE4 + KBEngine 开发的MMORPG游戏Demo，游戏规则基本复刻魔兽世界。

## 技术选型：

* 前端  ：UE        4.22
* 后端  ：KBEngine  2.5.7
* 数据库：mysql     8.0.17

建议围观的小伙伴采用和我相同的版本运行项目。
